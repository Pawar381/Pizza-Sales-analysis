# Pizza-Sales-analysis
Project overview
This project focuses on analyzing pizza sales data using SQL to extract meaningful business insights. It helps understand customer preferences, sales trends, and revenue generation by working with structured datasets.

The project is built using a relational database system and includes multiple tables such as orders, order details, pizzas, and pizza types.

🎯 Objectives
Analyze total number of orders and revenue
Identify top-selling pizzas and categories
Understand customer ordering patterns
Track sales performance over time
Support data-driven decision-making
🛠️ Technologies Used
SQL (MySQL)
Database Management System (DBMS)
📊 Key Features / Analysis Performed
Total number of orders placed
Total revenue generated from pizza sales
Highest-priced pizza identification
Most commonly ordered pizza size
Top 5 most ordered pizza types
Category-wise pizza sales analysis
Hourly distribution of orders (peak time analysis)
Average daily orders
Top 3 pizzas based on revenue
Percentage contribution of each pizza type
Cumulative revenue analysis using window functions
🧠 Concepts Used
SQL Joins
Aggregate Functions (SUM, COUNT, AVG)
Subqueries
Views
Group By & Order By
Window Functions
